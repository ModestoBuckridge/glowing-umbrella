
## Replit容器部署xray核心代理
### 提示：
1、本地上传项目时，名称勿出现xray/v2ray等任何的协议字样，大流量长期使用必被限制！

2、出现实例运行提示“You cant run this Repl right now”，请先尝试切换印度地区
